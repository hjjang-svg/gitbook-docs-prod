# 회원 가입

### 1.  [toss ads](https://tossads.toss.im/) 접속 후, 오른쪽 위 \[광고 시작하기] 를 눌러주세요

### 2. 로그인 화면에서 \[회원가입] 버튼을 눌러주세요

<figure><img src="../../.gitbook/assets/Group 228.png" alt=""><figcaption></figcaption></figure>

### 3. 회원가입을 위해 \[이메일주소]를 입력해주세요

<figure><img src="../../.gitbook/assets/이미지 (5) 1.png" alt=""><figcaption></figcaption></figure>

* 유의사항
  * 동일한 이메일 주소로는 1개의 계정만 생성할 수 있어요.
  * 이메일로 발송된 8자리 인증번호를 입력하는 인증 과정을 거쳐야 계정이 생성돼요.
  * 인증메일은 [toss-ads@toss.im](mailto:toss-ads@toss.im)로 발송되며, 수신 확인이 안되는 경우 스팸 메일함도 확인해주세요.
  * 비밀번호는 영문, 숫자, 특수문자가 모두 들어간 8\~16글자로 설정해주세요.



### 4. 본인 인증을 진행해주세요

<figure><img src="../../.gitbook/assets/애즈 비통과 룩 통일 1.png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/2-1) 세로형 영상 깃북 (2).png" alt=""><figcaption></figcaption></figure>

* 토스를 통해 본인 인증을 진행해요.
* **토스 회원이 아니라면, 토스에 가입해주세요.**
  * **왜 토스에 가입해야 하나요?** ([공지사항 확인하기](https://tossads.toss.im/notice/4981?tab=231))

### 5. 회원가입이 완료 되었다면 [비즈니스 그룹 생성](create.md) 후 광고 계정 운영이 가능해요&#x20;



{% hint style="info" %}
### 기존 회원의토스 비즈니스 회원 전환

토스 비즈니스 회원이 되면 토스에서 각각 운영되던 쇼핑, 애즈 등의 서비스에  통합 ID 하나로 로그인이 가능해져요.

토스 애즈를 이용하시던 고객분들께는 토스 비즈니스 회원으로 전환에 대해 안내 드리고 있어요.

비즈니스 회원 전환 동의 및  간단한 본인 인증을 진행하시면 전환이 완료돼요.
{% endhint %}
