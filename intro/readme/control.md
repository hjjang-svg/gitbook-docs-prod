# 광고계정 이관

### 광고계정 보내기

광고계정을 다른 비즈니스 그룹으로 옮기는 기능이에요.

* 캠페인 집행 이력, 도구 설정, 구성원 정보는 그대로 유지돼요.
* 설정한 이관일 00:00을 기준으로&#x20;
  * 새 비즈니스 그룹에서 광고계정을 사용할 수 있어요.
  * 해당 시점 이후 집행한 광고비는 이관된 계정에서 과금돼요.

***

#### 신청 전 반드시 확인하세요

* [x] 기존 비즈니스 그룹과 보내기를 진행할 비즈니스 그룹의 정산 방식(선불/후불)은 서로 같아야 해요.

- 예시) 후불 광고계정 → 선불 비즈니스 그룹 이동 ❌

* [x] 광고계정의 계정장만 이관을 신청할 수 있어요.
* [x] 선불 정산 계정의 경우 자동 충전 설정을 해제해야 이관신청이 가능해요.
* [x] 이관 신청이 접수되면, 이관 완료 전까지는 신규 캠페인을 생성 할 수 없어요.

- 신청 전 생성한 캠페인은 이관 중에도 중단 없이 운영돼요.

* [x] 이관은 이관받는 비즈니스 그룹장의 승인까지 완료되어야 최종 확정돼요.
* [x] 이관 신청이 완료된 후에는 직접 취소할 수 없고, 받는 그룹이 이관 요청을 반려하는 경우에만 자동 취소돼요.

***

### 이관일 및 정산 기준

이관일은 계정을 보내는 담당자가 직접 선택할 수 있고, 선택한 날짜 00:00부터 아래 내용이 적용돼요.

이미 지난 날짜는 선택할 수 없고, 신청 후에는 이관일 수정 변경은 불가해요

* **정산 주체 변경 :** 이관일 이후 발생한 광고비는 이관된 계정으로 과금돼요.
  * 이전 사용분 → 이관 전 광고계정
  * 이후 사용분 → 이관 후 광고계정
* **소속 그룹 변경 :** 새 비즈니스 그룹에서 광고계정을 관리할 수 있어요.

※ 선택한 날짜의 00:00까지 승인이 완료되지 않으면, 신청은 자동 취소되고 광고계정은 기존 비즈니스 그룹에 그대로 유지돼요.

### 광고 집행 현황에 따른 이관 후 변화

#### Case 1. 집행 중인 광고가 있는 경우

1\) 후불 광고계정

| 구분              | 과금 주체     | 집행 중인 캠페인 | 신규 캠페인 생성 |
| --------------- | --------- | --------- | --------- |
| ① 이관 신청 전       | 기존 광고계정   | 정상 집행     | 가능        |
| ② 신청 후 \~ 이관일 전 | 기존 광고계정   | 정상 집행     | 불가        |
| ③ 이관일 00:00     | 소속 변경 시점  | 정상 집행     | 불가        |
| ④ 이관일 이후        | 이관받은 광고계정 | 정상 집행     | 가능        |

* 광고는 중단되지 않아요.
* 이관일을 기준으로 사용 금액이 나뉘어 정산돼요.
  * 이전 사용분 → 기존 광고계정
  * 이후 사용분 → 이관받은 광고계정
* 단, 리워드 광고는 기존 광고계정에서 100% 정산돼요.

2\) 선불 광고계정

| 구분              | 과금 주체     | 집행 중인 캠페인       | 신규 캠페인 생성 |
| --------------- | --------- | --------------- | --------- |
| ① 이관 신청 전       | 기존 광고계정   | 정상 집행           | 가능        |
| ② 신청 후 \~ 이관일 전 | 기존 광고계정   | 정상 집행           | 불가        |
| ③ 이관일 00:00     | 소속 변경 시점  | 일시 정지           | 불가        |
| ④ 이관일 이후        | 이관받은 광고계정 | 광고머니 충전 시 자동 재개 | 가능        |

* 선불 계정은 이관 시점에 광고가 일시 정지돼요.
* 새 비즈니스 그룹에서 광고머니를 충전하면 자동으로 다시 집행돼요.
  * 단, 리워드 광고는 중단되지 않으며, 기존 광고계정에서 100% 정산돼요.
* 기존 광고계정의 잔액은 이관일 다음 날 오전 6시경 자동 환불돼요.

{% hint style="warning" %}
**리워드 광고 (공통)**

* 리워드 광고는 광고계정의 정산 방식(선불/후불) 여부와 관계없이, 이관 전·후에도 중단 없이 계속 집행돼요.
* 집행 중이거나 예약된 리워드 광고가 있는 경우, 해당 광고의 정산은 기존 광고계정에서 100% 진행돼요.
  * 이관 전 광고를 중단하거나, 이관 전·후 광고비를 나누어 정산하려면 고객센터로 문의해 주세요.
{% endhint %}

#### Case 2. 집행 중인 광고가 없는 경우

* 이관일 00:00부터 광고계정은 새 비즈니스 그룹으로 이전돼요.
* 같은 시점부터 새 비즈니스 그룹에서 캠페인을 바로 집행할 수 있어요.
* 이관일 이후 발생한 광고비는 새 광고계정의 정산 정보 기준으로 청구돼요.

### 자주 묻는 질문

**Q. 이관일이 지났는데 승인을 못했어요.**

A. 설정한 이관일 00:00까지 승인되지 않으면 신청은 자동 취소돼요.&#x20;

이 경우, 새로운 이관일을 설정해 다시 신청해 주세요.

**Q. 이관 후 세금계산서는 어떻게 발행되나요?**

A.이관일 00:00 기준으로 사용 금액이 나뉘어 발행돼요.

• 이전 사용분 → 이관 전 광고계정

• 이후 사용분 → 이관 후 광고계정

**Q. 이관 신청을 취소할 수 있나요?**

A.신청 후에는 직접 취소할 수 없어요. 승인 단계에서 반려되면 자동으로 취소돼요.

**Q. 선불 잔액은 언제 환불되나요?**

A.설정한 이관일 다음 날 오전 6시경 자동 환불돼요.환불 시간은 시스템 상황에 따라 달라질 수 있어요.

**Q. 리워드 광고 취소 위약금은 누가 부담하나요?**

A.이관 완료 후 리워드 광고를 취소하면, 위약금은 이관을 받은 광고계정에서 부담하게 되니 이관 신청 전 반드시 확인해주세요.

**Q. 머니알림 예약금은 어떤 계정으로 환불되나요?**

A.머니알림은 캠페인 집행 당일에 예약금이 환불돼요. 이 금액은 기존 광고계정으로 환불돼요.
