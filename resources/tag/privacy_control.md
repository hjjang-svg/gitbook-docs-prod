# 전환 추적 코드 공유/이관하기

특정 광고계정에서 생성한 전환 추적 코드를 다른 광고계정으로 공유하거나 소유권을 이관할 수 있어요.

* 전환 추적 코드는 **광고계정장만** 공유 또는 이관 설정이 가능해요.
* **소유권을 보유한 광고계정**에 한해 공유 또는 이관이 가능해요.

***

### 1. 공유하기&#x20;

다른 광고계정에서 동일한 전환 추적 코드를 **함께 사용**할 수 있도록 공유할 수 있어요.



1. 광고계정에 접속 후 **광고 도구** → **전환 및 추적 연동**을 클릭해 주세요.
2. 공유할 전환 추적 코드를 선택해 주세요.
3. **\[사용 중인 광고계정]** 영역 우측의 **\[추가]** 버튼을 클릭해 주세요.
4. 공유할 광고계정 정보를 입력해 주세요.
   * 광고계정 이름
   * 광고계정의 사업자등록번호

{% hint style="info" %}
공유한 광고계정은 자산 사용 목록에서 삭제할 수 있어요.\
삭제하더라도 이미 운영 중인 광고에는 해당 전환 추적 코드가 유지되며,삭제 이후에 생성하는 신규 광고에서는 사용할 수 없어요.
{% endhint %}

***

### 2. 이관하기&#x20;

전환 추적 코드의 **소유권을 다른 광고계정으로 이전**할 수 있어요.

1. 광고계정에 접속 후 **광고 도구** → **전환 및 추적 연동**을 클릭해 주세요.
2. 이관할 전환 추적 코드를 선택해 주세요.
3. **\[사용 중인 광고계정]** 하단의 **\[소유] 광고계정명** 우측 **\[수정]** 버튼을 클릭해 주세요.
4. 이관할 광고계정의 정보를 입력해 주세요.
   * 광고계정 이름
   * 광고계정의 사업자등록번호
   *   소유권 변경 이후에도 기존 광고계정에서 계속 사용할지 여부

       선택 시, 기존 광고계정은 **\[공유] 권한**으로 해당 전환 추적 코드를 계속 사용할 수 있어요.
