# 성과 대시보드

## 1. 성과 조회&#x20;

**기본 성과 조회**

* 대시보드에서 배너 광고 캠페인/세트/소재 단위로 광고 집행 기간 동안의 성과를 확인할 수 있어요.
  * **성과 데이터는 1시간 단위**로 업데이트되며 기본 날짜는 오늘로 선택되어 있어요.

<figure><img src="../../.gitbook/assets/일별 성과 조회 (1).png" alt=""><figcaption></figcaption></figure>

대시보드에서 사용할 수 있는 기타 유용한 기능들이 궁금하다면 [대시보드 유용한 기능](data.md#id-5) 을 참고해주세요.

***

* [x] **일별 성과 조회**

- **일별 상세** 보기 버튼을 누르면, 캠페인 광고 집행 기간 내에 성과를 일별로 확인할 수 있어요.
  * 일별 성과 조회는 최대 30일까지 조회 가능해요.

<figure><img src="../../.gitbook/assets/일별 성과 조회 (2).png" alt=""><figcaption></figcaption></figure>

* [x] **시간대별 성과 조회**

- 최근 30일 기간의 시간대 별 노출 수, 클릭 수, 도달 수, 재생 수 등의 성과 데이터를 볼 수 있어요.
  * 선택한 캠페인 수, 데이터 양이 많은 경우 다운로드에 실패할 수 있어요 실패하는 경우 선택한 범위를 줄여 시도하는 것을 권장해요.

<figure><img src="../../.gitbook/assets/시간대별 성과 조회 (1).png" alt=""><figcaption></figcaption></figure>

* [x] **잠재고객 성과 조회**&#x20;

- 잠재고객 모으기를 통해 수집된 리드는 아래 이미지와 같이 캠페인 목록에서 바로 다운로드 가능해요.
  * **잠재고객 수가 1건 이상 존재해야 \[잠재고객]** 버튼이 활성화되어 다운로드 할 수 있어요.

<figure><img src="../../.gitbook/assets/잠재고객 성과 조회 (1).png" alt=""><figcaption></figcaption></figure>

아래 이미지와 같이 여러 개의 캠페인을 선택하고 하단 \[잠재고객] 버튼을 눌러 원하는 캠페인 성과를 한 번에 다운로드할 수 있어요.

<figure><img src="../../.gitbook/assets/잠재고객 한 번에 다운 받기 (1).png" alt=""><figcaption></figcaption></figure>

## 2. 성과 다운로드하기

성과 다운로드는 집행이 종료된 날짜 기준 1일 이후 가능해요.

* [x] **단일 성과 다운로드**

- 캠페인/세트/소재 단위로 자유롭게 선택하여 성과를 다운로드 할 수 있고 엑셀 파일로 제공해요.
  * &#x20;경로: 배너 성과 탭 → \[다운로드] 버튼 클릭 → \[성과] 버튼 클릭&#x20;

<figure><img src="../../.gitbook/assets/단일 성과 다운로드 (1).png" alt=""><figcaption></figcaption></figure>



* [x] **다건 성과 다운로드**

- 여러 개의 캠페인을 선택한 후 하단에 위치한 \[성과] 버튼을 클릭하면 여러 개의 성과를 한 번에 다운로드 받을 수 있어요.
  * 배너 세트/ 소재 단위에서도 여러 개의 성과를 선택하여 한 번에 다운로드 할 수 있어요.

<figure><img src="../../.gitbook/assets/다건 성과 다운로드 (1).png" alt=""><figcaption></figcaption></figure>



* [x] **성과 다운로드 내역 조회**&#x20;

- 배너 성과 대시보드 하단에 노출되는 성과 다운로드 팝업은 최근 7일 동안 다운로드 했던 내역을 볼 수 있어요.
  * 성과 다운로드 내역은 7일 이후에 자동으로 삭제되고 따로 저장되지 않아요.

<figure><img src="../../.gitbook/assets/성과 다운로드 내역 조회 (1).png" alt=""><figcaption></figcaption></figure>

## 3. 성과 시간·일별 비교

* **캠페인 / 세트 / 소재  중**   1개 단위에서 최대 3개의 항목을 선택하여 비교 분석 그래프를 생성할 수 있어요.
  *   시간·일별 비교 대시보드에서 확인 가능한 지표는 아래와 같아요.

      * **클릭 수**&#x20;
      * **CPC**
      * **CPM**
      * **CTR**&#x20;
      * **재생 수**
      * **노출 수**
      * **잠재 고객 수**
      * **상세페이지 조회**
      * **앱 설치**
      * **회원가입**
      * **장바구니 담기**
      * **구매 완료**

      ※ 지표는 **최대 3개** 까지 선택할 수 있고 비교를 원하는 지표만 볼 수 있도록 제외하거나 추가 가능해요.

<figure><img src="../../.gitbook/assets/비교 분석 및 추가 (1).png" alt=""><figcaption></figcaption></figure>

* 비교 데이터는 **시간대 별 / 일별 단위**로 구분하여 볼 수 있고 시간·일별 비교 데이터를 제공하는 기간은 아래와 같아요.
  * 시간대 별 : 최대 7일 이내 기간 선택
  * 일별 : 최대 90일 이내 기간 선택

## 5. 대시보드 유용한 기능

* [x] **열 구성요소 변경**

- \[열 구성] 버튼을 눌러 대시보드에서 확인하고 싶은 열 구성요소들을 변경할 수 있어요.
  * 토스애즈는 노출/클릭/재생/잠재고객/전환 5 가지의 열 구성 설정을 기본으로 제공하고 있어요.
  * &#x20;열 구성을 설정 변경하지 않았다면 **전체** 열을 다 보여줘요.

<figure><img src="../../.gitbook/assets/열 구성_추가 (1).png" alt=""><figcaption></figcaption></figure>



* [x] **사용자 설정**

- 사용자 설정 아래 \[추가] 버튼을 누르면 아래와 같은 창에서 열 구성을 변경할 수 있고 열 순서도 드래그하여 조정할 수 있어요.

<figure><img src="../../.gitbook/assets/열 구성_추가 사용자 설정.png" alt=""><figcaption></figcaption></figure>

* [x] **열 너비 조정**

-   대시보드 각 열의 너비를 자유롭게 조절할 수 있어요.

    * 변경한 너비는 따로 저장하지 않아도 새로 고침하거나 페이지를 나가도 유지되어요.

    <figure><img src="../../.gitbook/assets/열 너비 조정 (1).png" alt=""><figcaption></figcaption></figure>



* [x] **정보 수정**

- **페이지 전환 없이 대시보드에서 바로** 캠페인/광고세트/소재의 정보들을 수정할 수 있어요.
  * 캠페인에서 수정가능한 정보는 다음과 같아요.
    * 캠페인명
    * 총 예산
    * 집행 기간
    * 소진 방법 (일반/빠른)
  * 세트에서 수정가능한 정보는 다음과 같아요.
    * 광고세트명&#x20;
    * 입찰 금액
    * 노출 기간
    * 노출 시간대 (요일별 설정 제외)
  * 소재에서 수정가능한 정보는 다음과 같아요.
    * 소재명

<figure><img src="../../.gitbook/assets/페이지 전환 없이 수정 (1).png" alt=""><figcaption></figcaption></figure>

* [x] **정렬**

- 정렬 기능을 제공하는 기준은 다음과 같아요.
  * 소진 비용
  * 노출 수
  * 클릭 수
  * CTR
  * 잠재고객 수
  * 재생수
  * 상세페이지 조회
  * 앱 설치
  * 회원가입
  * 장바구니 담기
  * 구매 완료

<figure><img src="../../.gitbook/assets/정렬 가능 버튼 (1).png" alt=""><figcaption></figcaption></figure>



* [x] **임시저장 캠페인 조회**&#x20;

- 임시 저장된 캠페인은 배너 광고 대시 보드 상단 \[임시저장] 탭에서 확인할 수 있어요.

<figure><img src="../../.gitbook/assets/임시저장 캠페인 조회 (1).png" alt=""><figcaption></figcaption></figure>
