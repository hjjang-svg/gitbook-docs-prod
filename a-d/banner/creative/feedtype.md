# 이미지 강조형 배너

<figure><img src="../../../.gitbook/assets/피드타입_영역 안내.png" alt=""><figcaption></figcaption></figure>

* [x] 광고 유형 : 리스트 배너 → 소재 유형 **\[이미지 강조]** 선택&#x20;

- 소재 심사 규정에 따라  **평어체 (반말)** 를 사용할 수 없고 맞춤법에 어긋나는 표현 혹은 신조어는 포함되지 않아야 해요.
- [업종 별 심사 가이드](https://toss-ads.gitbook.io/guide/policy/industryguide#id-1)에 따라 심의필 제출이 필수 업종인 경우, 카테고리를 선택하고 안내 문구 기재란에 작성해주시면 하단 영역에 자동으로 반영돼요.

{% tabs %}
{% tab title="광고 이미지" %}
* **광고 이미지**&#x20;
  * 1200\*675 사이즈 jpg, png 이미지 (비율 : 16:9)
  * 파일 크기 : 5MB 이하
  * 이미지 내에 문구가 30% 이상 차지하는 경우 심사 반려될 수 있어요.
    * 정보 전달성 문구는 이미지 속 문구 입력란을 활용해주세요.
  * **투명 배경, 강한 형광색 컬러의 배경 이미지는 사용할 수 없어요.**
* **이미지 속 문구**&#x20;
  * 이미지 속 문구는 **선택 사항**이고 주요 문구와 보조 문구로 구성되어 있어요.
  * 본문에 포함된 문구가 50% 이상 중복되는 경우 심사 반려될 수 있어요.

<figure><img src="../../../.gitbook/assets/피드타입_이미지 속 문구 good&#x26;bad (2).png" alt=""><figcaption></figcaption></figure>

* **이미지 속 주요문구** : 공백 포함 28자 입력 가능하고 줄바꿈이 가능해요. (한 행에 최대 14자 입력 가능)&#x20;
* **이미지 속 보조 문구(선택)** : 공백 포함 최대 16자 입력 가능해요.
* **광고 문구색 :** 블랙과 화이트 중에서 선택 가능해요.
  * 소재 심사 규정에 따라 모든 문장은 **명사형** 또는 **\~해요체**로 기재해야 하고 초성으로 기재할 수 없어요.
{% endtab %}

{% tab title="광고 문구" %}
* **주요 문구**
  * 공백 포함 최대 18자 입력 가능해요
* **보조 문구**
  * 공백 포함 최대 18자 입력 가능해요
* 주요 문구와 보조 문구에는 **중복된 혜택/ 정보를 기재할 수 없어요**
  * 가독성을 위해 서로 다른 정보 / 혜택을 기재하거나 보조할 수 있는 문장으로 구성해주세요
    * 광고 텍스트 색상 변경 옵션을 제공하지 않아요
* 소재 심사 규정에 따라 모든 문장은 **명사형** 또는 **\~해요체**로 기재해야 하고 초성으로 기재할 수 없어요
* **버튼(cta)**
  * 버튼 문구는 아래 옵션 내에서 활용할 수 있어요
    * 더 알아보기, 확인하기, 보러가기, 구경하기, 자세히 보기, 구매하기, 사러가기, 설치하기, 다운로드, 받기, 앱 열기

<figure><img src="../../../.gitbook/assets/피드타입_CTA (1).png" alt=""><figcaption></figcaption></figure>
{% endtab %}

{% tab title="브랜드 명,로고" %}
* **브랜드명**
  * 공백 포함 최대 20자 입력 가능해요.
* [브랜드 이미지](logo.md) 설정 가이드를 참고하여 이미지를 설정해주세요.
  * 로고 이미지가 실제 노출 지면에 반영되었을 때 **잘림 또는 왜곡 없이 잘 보여지도록 이미지 여백을 충분히** 넣어주세요.

<figure><img src="../../../.gitbook/assets/로고,제품 안전 여백 (6).png" alt=""><figcaption><p>로고 이미지 안전 여백 가이드</p></figcaption></figure>
{% endtab %}

{% tab title="연결 URL" %}
* [**유저가 접속할 때 이상이 없어야 해요**](../../../policy/creativeguide.md#id-5)
  * 앱으로 랜딩된다면, 앱 미설치 유저는 앱 스토어 / 구글 플레이스토어로 랜딩될 수 있어야 해요
  * 랜딩 과정에서 접속하는 데 5초 이상 걸리는 경우, 브릿지 페이지를 만들어 주세요
* [**랜딩/이벤트 페이지의 내용과 소재 내용이 일치해야 해요**](../../../policy/creativeguide.md#id-6)
  * 랜딩 / 이벤트 페이지에서 확인할 수 없는 혜택, 정보는 광고 승인이 불가해요
  * 가입 또는 응모와 같이 선행 조건이 따르는 혜택은 광고 문구 내에 반드시 명시해야만 사용할 수 있어요
* [**URL 주소는 반드시 영문으로 제출해주셔야 해요**](../../../policy/creativeguide.md#id-4)
  * 주소에 한글이 있는 경우 오류가 발생할 수 있어요
  * 링크 내에 <mark style="color:red;">{ } ^ #</mark> 문구가 있으면 반려될 수 있어요
    * <mark style="color:red;">^</mark> 대신 <mark style="color:red;">\_</mark>(언더바)를 사용해주세요
  * <mark style="color:red;">http://</mark> 는 사용이 불가하여 <mark style="color:red;">https://</mark> 를 사용해주세요
* 앱스플라이어로 생성한 원링크를 제출할 경우, 파라미터가 단축되지 않은 원본 롱링크를 제출해 주세요
* 숏링크/단축 URL 형태는 사용할 수  없어요 (비틀리 포함)
* 이 외의 사유로 CTA 링크에 이슈가 있다고 판단되는 경우, 링크 수정 요청 할 수 있어요
* 그 외 공통으로 적용되는 규정은 [소재 심사 규정](../../../policy/creativeguide.md#id-4) 을 참고해주세요
