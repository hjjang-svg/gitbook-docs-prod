# 광고 유형별 소재 가이드

> 원활한 광고 운영 및 심사를 위해 소재 작성 시에는 **소재 공통 가이드**와 **광고 유형 별 소재 가이드**를 반드시 참고해 주세요.

***

### &#x20;광고 심사 신청   <a href="#common" id="common"></a>

<figure><img src="../../../.gitbook/assets/캠페인 전체 심사 신청 (1).png" alt=""><figcaption></figcaption></figure>

* **작성한 광고는 아래 두 가지 방법 중 하나를 선택하여, 효율적으로 심사 신청할 수 있어요.**
  * <mark style="background-color:orange;">**전체 심사 신청**</mark>**&#x20;:** 캠페인 하위에 제출 가능한 소재는 모두 일괄 심사 요청돼요.
  *   <mark style="background-color:orange;">**이 소재 심사 신청**</mark>**&#x20;:** 심사 신청이 필요한 소재만 개별 심사 요청돼요.

      * 세트 단위로 심사 신청은 불가해요.


* **캠페인 / 세트 / 소재를 생성하고 심사 제출하지 않아도&#x20;**<mark style="background-color:orange;">**임시(자동) 저장**</mark>**되어요.**
  * 임시 저장된 캠페인은 가장 마지막에 수정된 내용으로 14일 동안 보관되고, 14일 동안 수정한 내용이 없는 경우 자동 삭제돼요.
  *   임시 저장된 캠페인은 아래 경로에서 언제든지 수정할 수 있어요.

      * 배너 →  캠페인 대시 보드 → 임시 저장 캠페인 명 우측 '상세' 버튼 클릭


* **심사 신청 후 반려된 소재는 자동 저장되지 않고&#x20;**<mark style="background-color:orange;">**'저장하기'**</mark>**&#x20;버튼을 눌러 수정 사항을 저장할 수 있어요.**

<figure><img src="../../../.gitbook/assets/반려 소재 저장하기 및 이 소재 심사 신청 (1).png" alt=""><figcaption><p>반려 소재 저장하기 / 이 소재 심사 신청</p></figcaption></figure>

*   **캠페인 기간이 남아 있다면 기간 내에 광고 세트와 소재를 추가하고 심사를 요청할 수 있어요.**

    * 캠페인 종료 당일 / 소재 심사 중 상태에서는 소재 수정이 불가해요.


* &#x20;**캠페인당 광고세트 100개, 광고세트당 소재 100개 까지 생성 가능해요.**
  * 위 제한 사항보다 갯수를 초과하여 생성했던 캠페인은 더 이상 광고세트와 소재를 추가할 수 없도록 추가 버튼이 비활성화 돼요.
  * 추가로 소재를 세팅해야 하는 경우 새로운 캠페인을 생성하여 세팅해주세요.

### &#x20;캠페인 운영 기능 <a href="#common" id="common"></a>

* 광고 세트 및 소재 **on/off** 기능으로 직접 운영하실 수 있어요.
  * off 된 소재는 과금되지 않아요.
* 심사 승인된 소재는 삭제할 수 없어요.

## 소재 공통 가이드  <a href="#common" id="common"></a>

* 소재 검수 시간은 평일 **10:00 \~ 19:00**이며, 모든 소재는 순차적으로 검수되고 있어요.
  * <mark style="color:blue;">원활한 소재 검수 및 광고 캠페인 운영을 위해 반드시 집행일 기준</mark> <mark style="color:blue;"></mark><mark style="color:blue;">**2-3영업일 전**</mark> <mark style="color:blue;"></mark><mark style="color:blue;">소재를 제출해 주세요.</mark>

{% hint style="danger" %}
**광고 상품 별 상세 가이드는 하위 항목을 참고해주세요.**
{% endhint %}

{% tabs %}
{% tab title="문구" %}
* 광고 문구 색상 변경 옵션을 제공하지 않아요.



* **평어체 (반말)**&#xB97C; 사용할 수 없고 맞춤법에 어긋나는 표현 혹은 신조어는 포함되지 않아야해요.



* 모든 소재의 문구는 **최소 5글자 이상** 작성해야 해요.



* 토스 기존 서비스와 문구 톤을 맞춰야 광고 효율 (클릭률)이 올라 갈 수 있어요.



* 중복되거나 유사한 내용의 소재를 반복적으로 등록하는 경우 광고 집행이 제한될 수 있어요.
{% endtab %}

{% tab title="특수 문자" %}
<mark style="background-color:red;">**과도한 강조, 과장 표현으로 오해될 수 있거나 불필요한 특수 문자 사용을 제한**</mark><mark style="background-color:red;">하고 있어요.</mark>

* 사용 가능한 특수 문자는 다음과 같아요&#x20;
  * &#x20; <mark style="color:red;">**% & - + = · ? / → ‘ ’ " " \~ ! ( ) , . \[]**</mark>
* **사용 시 유의 사항**
  * **허용된 특수 문자라 하더라도 반복 사용은 제한하고 있어요.**\
    (예: `!!!`, `~~`, `??`, `%%`  )
  * **특수 문자로 이어지는 정보 단순 나열 형식의 구성은 광고 효율을 위해 권장하지 않아요.**
  * **온점(.)은 각줄 문장의 끝에 사용할 수 없어요.**
* 특수 문자를 사용하여 광고 소재를 공지,알림 정보로 오인할 수 있는 문장으로 구성하는 경우 심사 승인이 불가해요.&#x20;
* <mark style="color:blue;">**사용 가능 예시**</mark>
  * 최대 10<mark style="color:purple;">%</mark> 할인 &#x20;
  * 지금 가입하면 혜택까지<mark style="color:purple;">!</mark>
  * 스킨 케어 BEST 1<mark style="color:purple;">+</mark>1 혜택&#x20;
* <mark style="color:red;">**사용 불가 예시**</mark>
  * 지금이<mark style="color:red;">!!</mark> 기회입니다<mark style="color:red;">! →</mark>&#x20;
  * 가입하면 <mark style="color:red;">·</mark> 혜택이<mark style="color:red;">? !</mark>&#x20;
  * <mark style="color:red;">(</mark>알림<mark style="color:red;">)</mark>즉시 수령 가능 현금<mark style="color:red;">+</mark>
{% endtab %}

{% tab title="이미지" %}
* **토스 앱의 실제 화면 구성이나 기능, 알림처럼 보이는 이미지**를 사용하는 경우, 광고를 접한 유저가 **공식 기능으로 오인하거나 앱 내부 콘텐츠로 착각할 수 있어** 사용을 제한하고 있어요.

<figure><img src="../../../.gitbook/assets/image 150.png" alt=""><figcaption></figcaption></figure>

* 사용 불가 예시&#x20;
  * 토스에서 제공하는 서비스 및 알림 아이콘과 유사한 경우
  * 토스 앱과 유사한 UI구성을 캡처하거나 그대로 재현한 경우
  *   브랜드와 연관되지 않은 붉은 톤, 고채도 색상을 사용한 경우&#x20;


* 토스와 공식적으로 제휴하거나 사전 협의가 이루어진 광고가 아닌 경우 **토스의 브랜드명** 또는 **브랜드 로고**를 사용할 수 없어요.<br>
* 소재 심사 규정에 따라 토스 앱 배경 색상과 같은 컬러의 배경 이미지 또는 지나치게 채도가 높은 형광색 이미지는 사용할 수 없어요 [상세 가이드](../../../policy/creativeguide.md#id-2) 를 참고하여 이미지를 구성해주세요.
{% endtab %}

{% tab title="랜딩 페이지" %}
* 랜딩 페이지에서 확인할 수 없는 **혜택 및 정보**는 광고 소재로 사용할 수 없어요.
  * 오류 페이지 또는 접근 불가한 랜딩 페이지로 확인되거나 이벤트 오픈 전 랜딩 URL은 1차 반려돼요.
  * 랜딩 페이지가 구성되기 전에 광고 심사 신청하는 경우 심사에 참고할 수 있는 증빙 자료를 제출해주세요. <br>
* _가입_  또는 _응모_ 와 같이 **선행 조건이 따르는 혜택**은 광고 문구 내에 반드시 명시해야만 사용할 수 있어요.
  * <mark style="background-color:orange;">예시 )</mark>&#x20;
    * <mark style="color:orange;">토스애즈 가입하고 1만원 혜택 받아요</mark>
    * <mark style="color:orange;">경품 응모하면 1만 포인트 증정</mark><br>
* 광고 문구 내에 기재된 정보가 허위 사실로 밝혀질 경우 광고 집행이 불가해요.
  * 이미 승인된 소재라도 광고 집행 제한 또는 집행 중단 될 수 있어요.<br>
* 토스와 공식적으로 제휴하거나 사전 협의가 이루어진 광고가 아닌 경우 랜딩 페이지 내에 **토스의 브랜드명 또는** **서비스명**을 사용할 수 없어요.
{% endtab %}
