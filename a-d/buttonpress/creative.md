# 소재 가이드

## 버튼 누르기 소재 만들기

> 소재 검수 시간은 평일 **10:00 \~ 19:00**이며, 모든 소재는 순차적으로 검수되고 있어요.
>
> <mark style="color:blue;">원활한 소재 검수 및 광고 운영을 위해서 집행일 기준</mark> <mark style="color:blue;"></mark><mark style="color:blue;">**2-3영업일 전**</mark> <mark style="color:blue;"></mark><mark style="color:blue;">소재를 제출해주세요.</mark>

### 1. 광고 집행에 필요한 소재 리소스

**버튼 누르기 광고를 집행하려면 아래 리소스가 필요해요.**

* **브랜드 로고 1종**
* **클릭 시 이동할 페이지 링크**

카테고리에 따라,\
소재 제출 시 **토스애즈 심의필 번호를 제출하고 소재에 함께 넣어야 해요.**\
자세한 내용은 **업종별 심사 가이드**에서 확인해 주세요.

#### 로고 소재 가이드

* **배경색이 포함된 800×800px 이상 PNG 파일**을 업로드해 주세요.
* **투명 배경 로고는 사용할 수 없어요.**
* **형광처럼 강한 색의 로고는 수정 요청이 있을 수 있어요.**

### 3. 랜딩 URL 가이드

유저가 버튼을 누르면 연결되는 **URL과 이벤트 페이지는 아래 규정을 지켜주세요.**

* 유저가 접속하는 데 문제가 없어야 해요.
* 앱으로 이동한다면, 앱이 없는 유저는 **앱 스토어 또는 구글 플레이스토어**로 이동해야 해요.
* 랜딩에 **5초 이상** 걸리면, **브릿지 페이지를 만들어 주세요.**

#### URL 정책

* 주소에 **한글이 있으면 오류가 날 수 있어요. 영문 URL을 사용해 주세요.**
* 링크에 **{ } ^ #** 문자가 있으면 승인되지 않아요.\
  → **^ 대신 \_(언더바)** 는 사용할 수 있어요.
* **http:// 는 사용할 수 없고, https:// 만 가능해요.**
* **UTM 파라미터가 포함된 원본 링크**로 제출해 주세요.
* **비틀리 같은 단축 URL은 사용할 수 없어요.**
* 앱스플라이어 링크도 **축약되지 않은 원본 롱링크**로 입력해 주세요.
* **UTM 파라미터가 포함된 원본 링크라면 딥링크 사용은 가능해요.**
* **단축된 딥링크는 사용할 수 없어요.**
* 이외에도 CTA 링크에 문제가 있다고 판단되면, **링크를 다시 요청할 수 있어요.**
