## **광고 생성하기**

{% stepper %}
{% step %}
### 광고 생성을 위해 각 항목에 맞는 정보를 입력해주세요

<figure><img src="../../.gitbook/assets/광고 생성하기 (3).png" alt=""><figcaption></figcaption></figure>

&#x20;✔ 상품명: **베이직 / 플러스 / 프리미엄** 중 선택 가능해요.&#x20;

* 베이직(벤치: 5만 트래픽)&#x20;
* 플러스(벤치: 15\~20만 트래픽)&#x20;
* 프리미엄(벤치: 27만 트래픽)&#x20;

&#x20;✔ 방송 진행 시간 : 상품에 따라 선택 가능한 방송 진행 시간이 늘어나요.

* 베이직 :최소 20분\~최대 1시간 / 20분 단위로 선택 가능
* 플러스 : 최소 30분\~최대1시간 / 10분 단위로 선택 가능
* 프리미엄 : 최소 1시간\~최대 6시간 / 30분 단위로 선택 가능

✔ 방송 시작 일시 &#x20;

* 라이브 방송 시작 24시간 전 부터 토스 앱에 노출되어 유저에게 사전 알림이 발송돼요.
  * 방송 시작 당일 부킹 시 광고 효율을 보장하기 어려워요.
{% endstep %}

{% step %}
### &#x20;약관 동의 확인 및 체크&#x20;

<figure><img src="../../.gitbook/assets/약관 동의 확인 (2).png" alt=""><figcaption></figcaption></figure>

&#x20;✔ 광고 집행 시 필수적으로 확인 필요한 **위약금 정책** 및 **운영, 정산 관련** 내용을 안내하고 있어요 광고를 생성 하기 전 반드시 확인하고 동의 여부를 체크해주세요.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
&#x20;라이브 마켓은 논타겟 상품으로, 만 19세 이상 성인 타겟 설정 필요한 광고는 집행이 어려워요.

* 예시 ) 전자 담배, 주류, 청소년 관람불가 영상물 등
{% endhint %}
