# 소재 가이드

#### 행운퀴즈 **소재 설정 가이드**

<figure><img src="../../.gitbook/assets/행운퀴즈_영역 안내.png" alt=""><figcaption></figcaption></figure>

* 소재 공통 가이드에 따라 타이틀 및 상품 정보에 **평어체(반말)**&#xC744; 사용할 수 없어요.
* 랜딩 페이지에서 확인 불가한 혜택,정보는 광고 소재에 기재할 수 없어요.
  * 반드시 랜딩 URL(이벤트 페이지)에서 확인 가능한 정보로 광고 소재를 구성해주세요.
* 소재 검수 시간은 **평일 10:00 \~ 19:00**이며, 모든 소재는 순차적으로 검수되고 있어요.
  * 원활한 광고 운영을 위해서 집행일 기준 **2-3영업일 전** 소재를 제출해 주세요.

{% hint style="info" %}
- 방송 시작 일시는 1일 단위로 선택 가능하고 방송 시작 당일 소재 제출 시 광고 집행 가능 여부를 확정할 수 없어요.
  * 광고 효율을 위해 2-3 영업일 전 소재를 제출해주세요.
- 심사 승인 이후에도 광고 집행 일시가 지나지 않았다면 소재 수정 후 재심사 요청할 수 있어요.
  * **심사 중**이거나 **집행 중** 또는 **집행완료** 상태인 경우 소재 수정 불가
{% endhint %}

{% tab title="퀴즈 리스트" %}
* [x] **광고를 접하는 토스 앱 유저는 이 리스트를 통해 행운퀴즈에 진입해요.**

<figure><img src="../../.gitbook/assets/퀴즈 리스트.png" alt=""><figcaption></figcaption></figure>

* **리스트명**
  * 공백 포함 최대 12자까지 입력 가능해요.
  * 명사형으로 작성해야 하고 평어체 또는 초성을 사용할 수 없어요.
    * 진행 중인 기획전명, 프로모션 또는 퀴즈 내용을 요약할 수 있는 문구로 구성해주세요.
  * 퀴즈의 정답이 리스트 영역에서 노출되지 않아야 해요.
    * 리스트명에 정답 노출 시 유저가 힌트 페이지를 방문하지 않고 정답을 맞출 수 있어요.
* **브랜드 로고**
  * &#x20;배경색이 투명한 800\*800 사이즈 png 이미지
  * 로고만 삽입된 **정사각형 이미지**&#x20;
    * 제품 이미지 및 기획전 이미지 사용 불가&#x20;
* 강한 형광색, **토스의 배경 색상과 같은 컬러**의 배경 이미지는 사용할 수 없어요.

<figure><img src="../../.gitbook/assets/퀴즈 화면 이미지.png" alt=""><figcaption></figcaption></figure>

* 광고 효율을 위해 유저가 브랜드를 명확하게 인지할 수 있도록 반드시 공식 로고를 사용해주세요.
  * 브랜드 로고는 \[퀴즈 리스트],\[퀴즈 화면],\[참여 후 화면]에 노출돼요.
{% endtab %}

{% tab title="퀴즈 화면 이미지" %}
* [x] &#x20;**유저는 이 화면에서 퀴즈를 풀고 리워드를 받아요.**

- **대표 이미지**&#x20;
  * 1500\*840 사이즈 jpg, png,jpeg 이미지
- **이미지 색상 가이드**
- **퀴즈의 힌트 또는 정답이 될 수 있는 요소가 이미지 내에 포함되지 않아야 해요.**
  * 퀴즈의 정답이 대표 이미지에 노출되면 유저가 힌트 페이지 (랜딩 URL) 방문 없이 정답을 맞출 확률이 높아요 이로 인한 불이익 발생 시 토스에서 보상하지 않으니 반드시 확인해주세요.
- **이미지 내에 텍스트가 30% 이상이면 사용할 수 없고** _(이미지 전체를  3분할로 나눈 것 기준)_ **사용 불가한 세부 기준은 아래와 같아요.**
  * 그래픽 요소로 구성된 텍스트를 삽입한 경우 광고 소재에 기재된 혜택/ 정보와 중복 기재할 수 없어요.
    * 광고 효율과 가독성을 고려하여 동일한 내용이 반복하여 기재되지 않도록 구성해주세요.
  * 브랜드명 또는 로고 삽입이 필요한 경우 **좌/우 상단 중 한 곳에 \[140\*140px] or \[250\*76px]** 크기로 배치해주세요.
  * 이미지 내 피사체가 과도하게 많거나, 과도한 크롭 / 클로즈업으로 인해 주요 제품을 알아볼 수 없는 이미지는 사용 불가해요.
  * 3개 이상 복잡한 오브젝트로 구성되어 주요 제품을 알아 볼 수 없는 이미지는 사용 불가해요.
- **이미지 내에 충분한 여백 없이 전체 비중을 차지하는 경우 심사 반려될 수 있어요.**
  * 광고 집행 시 이미지 잘림이 발생할 수 있어 최소 20% 이상 여백을 두고 이미지 구성해주세요.
    * 단색 배경 외 상품 이미지, 텍스트, 꾸밈 요소, 배경 패턴 등은 여백으로 판단하지 않아요.
- **오브젝트가 과도하게 많거나 과도하게 크롭/클로즈업된 이미지는 사용 불가해요.**
  * 주요 제품을 강조하기 위해 최대한 가운데에 배치하여 구성해주세요.
- [**업종 별 심사 가이드**](https://toss-ads.gitbook.io/guide/policy/industryguide)**에 따라 심의필 제출이 필수인 업종의 경우, 전체 심의필 문구를 좌측 하단을 제외한 모서리 영역에 가독성 높게 기재해주세요.**

{% tab title="퀴즈 내용" %}
* [x] **정답이 될 단어는 숫자 또는 한글단어만 입력 가능해요.**
  * [x] 퀴즈 내용과 정답은 영문으로 기재할 수 없고 한글로 변역/치환하여 설정하는 경우 심사 승인이 불가해요.
* [x] 광고를 접한 유저가 힌트 페이지에서 **스크롤을 아래로 내리는 행위만으로 정답을 찾을 수 있도록** 구성되어야해요.

- **퀴즈 내용**
  * 최소 1개\~최대 3개까지 추가 가능해요.
  * 광고 효율을 위해 2개 이상 설정하는 것을 권장해요.
- **퀴즈 문장은 한 행당 공백 포함 최대 14자 입력 가능하고 3행 이하로 작성해주세요.**
  * ‘\~해요’체의 문장형으로 작성해주세요.
    * 문장이나 단어는 초성으로 기재할 수 없어요.
    * ‘\~습니다’체로 구성한 경우 심사 승인이 불가해요.
- **퀴즈 정답 단어는 공백 없이 8자 입력 가능해요.**
  * 정답이 될 단어를 **{}** 중괄호로 표시하면 정답으로 설정돼요.
    * <mark style="color:orange;">입력 예시) 대표적인 간편 송금 서비스인 {토스}를 지금 바로 만나보세요.</mark>
  * 정답이 될 단어는 **숫자** 또는 **한글** 단어만 입력할 수 있어요.
  * 힌트 페이지와 설정한 정답은 동일해야 해요.
  * 띄어쓰기(공백)가 포함된 정답은 사용할 수 없어요.
  * 퀴즈 정답은 서로 중복되지 않아야해요.
    * 예시로,1번 퀴즈와 3번 퀴즈 정답이 중복되는 경우 심사 반려될 수 있어요.
- **맞춤법에 어긋나는 표현 혹은 신조어는 포함되지 않아야해요.**
- **사용 가능한 특수 문자는 다음과 같아요. \[&#x20;**<mark style="color:red;">**% + ,**</mark>**&#x20;]**&#x20;
  * 그 외 특수 문자는 사용할 수 없어요.
- **심의 문구는 퀴즈 내용에 포함하여 기재할 수 없어요 퀴즈 화면 대표 이미지에 기재해주세요.**
{% endtab %}

{% tab title="힌트 페이지 URL" %}
* [x] **퀴즈 내용을 이해하기 어렵거나 힌트 페이지에서 정답을 찾기 어려운 경우 광고 부스팅 효과가 저하될 수 있으니 반드시 아래 설정 가이드에 따라 구성해주세요.**

- **퀴즈와 정답은 힌트 페이지에서 큰글씨, 볼드,하이라이트 등 직관적으로 확인할 수 있도록 구성해주세요.**
  * 심사 과정에 모바일 화면 기준 정답 문구가 너무 작거나 식별 불가한 경우 심사 반려될 수 있어요.
  * 랜딩 페이지가 길거나 정보가 많은 경우 **상단에 위치한 문구 활용**을 권장해요.
  * 직관적으로 찾기 어려운 페이지 하단에 위치해 있을 경우 랜딩 또는 퀴즈 정답 수정이 필요할 수 있어요.
  * 팝업, 움직이는 배너, 클릭을 통한 이동 등으로 확인 가능한 정답은 활용 불가해요.
- **\[상품 정보 더보기], \[상세 페이지 보기] 와 같은 버튼을 클릭하기 전에 노출된 영역만 힌트 페이지로 사용 할 수 있어요.**

* [x] **랜딩 URL 오기입 시 발생되는 불이익에 대해서는 토스에서 보상하지 않아요 광고 생성 시 문제 없이 사용 가능한 URL이 맞는지 반드시 확인해주세요.**

- [**유저가 접속할 때 이상이 없어야 해요**](https://toss-ads.gitbook.io/guide/policy/creativeguide#id-5)**.**
  * 행운퀴즈는 <mark style="color:red;">실시간 동시 접속이 분당 15만명 이상</mark> 발생할 수 있어요 이 점을 고려하여 단기간에 많은 트래픽 접속이 가능한 페이지로 설정해주세요.
  * 광고 노출 시간 동안 문제 없이 랜딩이 가능해야 해요.
  * 앱으로 랜딩된다면, 앱 미설치 유저는 앱 스토어 / 구글 플레이스토어로 랜딩될 수 있어야 해요.
  * 앱을 설치한 유저는 앱 안으로 연결되어야 해요.
  * 랜딩 과정에서 접속하는데 5초 이상 걸리는 경우, 브릿지 페이지를 만들어 주세요.
  * 모든 랜딩 페이지는 반드시 모바일 전용 웹/앱 페이지로 구성되어야 해요.

* [x] **광고 혜택 및 정보를 증빙할 수 있는 랜딩 페이지 구성이 불가한 경우 첨부의견을 통해 방송 혜택을 확인할 수 있는 증빙 자료를 제출해주세요. ( 랜딩 시안,목업 및 방송 스케줄 정보 등)**

- [**랜딩/이벤트 페이지의 내용과 소재 내용이 일치해야 해요**](https://toss-ads.gitbook.io/guide/policy/creativeguide#id-6)**.**
  * 랜딩 / 이벤트 페이지에서 확인할 수 없는 혜택, 정보는 광고 승인이 불가해요.
  * 정답 또는 힌트를 확인하기 위해 제약 조건(**로그인 혹은 회원가입**) 선행되는 랜딩페이지는 사용이 불가해요.
  * 랜딩 페이지 수정이 어렵다면, 팝업이나 고정된 배너를 활용하여 광고 소재에 기재한 혜택 정보를 기재해 주세요.
  * 랜딩 페이지 내에는 접속한 유저가 상품/이벤트/프로모션에 대해 문의할 수 있는 문의처가 기재되어 있어야 해요.
- **토스와 공식적으로 제휴하거나 사전 협의가 이루어진 광고가 아닌 경우 랜딩 페이지 내에 토스의 브랜드명 또는 서비스명을 사용할 수 없어요.**
  * 내부 협의가 완료된 경우 증빙 자료를 제출해주세요.
- [**URL 주소는 반드시 영문으로 제출해주셔야 해요**](https://toss-ads.gitbook.io/guide/policy/creativeguide#id-4)**.**
  * 주소에 한글이 있는 경우 오류가 발생할 수 있어요.
  * 링크 내에 { } ^ # 문구가 있으면 반려될 수 있어요.
    * ^ 대신 \_(언더바)를 사용해주세요.
  * http:// 는 사용이 불가하여 https:// 를 사용해주세요.
  * 숏링크/단축 URL 형태는 사용할 수 없어요. (비틀리 포함)
    * 앱스플라이어로 생성한 원링크 기입 시에도 축약형 숏링크가 아닌 원본 롱링크를 기입 해주세요.
- 이외의 사유로 CTA 링크에 이슈가 있다고 판단되는 경우, 링크 수정 안내 드릴 수 있어요.
  * 그 외 공통으로 적용되는 규정은 [소재 심사 규정](https://toss-ads.gitbook.io/guide/policy/creativeguide#id-4) 을 참고해주세요.
{% endtab %}

{% tab title="퀴즈 참여 후 혜택 타이틀 " %}
* [x] 퀴즈를 풀고 리워드를 받은 후 보여지는 화면이에요 유저는 이 화면에서 이벤트 페이지를 방문할 수 있어요.

- **혜택 타이틀**&#x20;
  * **브랜드명** 또는 **브랜드명+제품 카테고리** 외에 다른 정보를 입력할 수 없어요.
  * 초성 또는 특수문자를 입력하는 경우 심사 승인이 불가해요.
- **이벤트 페이지 URL**&#x20;
  * [힌트 페이지 ](creative.md#url)가이드와 동일하게 광고 집행 시간 동안 문제 없이 유저가 접속 가능해야 해요.
    * 그 외 상세 가이드도 힌트 페이지와 동일해요.
