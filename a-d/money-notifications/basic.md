# 일반형

<figure><img src="../../.gitbook/assets/머니알림 일반형_영역 안내 (1).png" alt=""><figcaption></figcaption></figure>

* **소재 공통 가이드에 따라 메인/서브 카피에 평어체(반말)을 사용할 수 없어요.**
* **랜딩 페이지에서 확인 불가한 혜택 및 이벤트 정보는 소재 내에 기재할 수 없어요. 반드시 연결되는 랜딩 페이지에서 확인 가능한 정보로 소재를 구성해주세요.**
* 소재 검수 시간은 **평일 10:00 \~ 19:00**이며, 모든 소재는 순차적으로 검수되고 있어요.

{% hint style="info" %}
- 소재  심사 가이드에 따라 심사 기간이 소요될 수 있어 광고 집행일 기준 2-3 영업일 전 소재를 제출해주세요. 광고 집행일 기준 1영업일 전 또는 몇 시간 전 제출된 소재는 광고 집행 전까지 승인을 보장할 수 없어요.
- 집행 상태&#xAC00;**\[집행 예정]**, \[**집행 중], \[집행 완료], \[집행 중단]** 이거나 소재 상태가 **\[심사중] ,\[승인]** 상태인 경우에는 소재 수정이 불가해요.
  * 방송 혜택이 변경된 경우에만 내부 심사 및 정책 기준에 따라 예외적으로 수정 도움 드리고 있어요.
- 그 외 소재 수정이 필요한 경우 소재 집행 중단 처리 후 소재를 새롭게 제출해주세요.
  * **집행 중단 일자에 따라 위약금이 부과될 수 있으니 유의해주세요.**
{% endhint %}

## 1. 이벤트 소개 영역

<figure><img src="../../.gitbook/assets/이벤트 소개 영역 (4).png" alt=""><figcaption></figcaption></figure>

{% tabs %}
{% tab title="메인 타이틀" %}
유저가 흥미를 가질 수 있는 상품 혹은 이벤트 **혜택 단 하나**만 강조하여 한 문장으로 작성해주세요.

* ‘\~해요’체의 문장형으로 작성해주세요.
  * 문장이나  단어는 초성으로 기재할 수 없어요.
* 문장은  최소 2행, 최대 3행까지 입력 가능해요.
  * 한 행당 공백 포함 최대 18자 입력 가능해요.
* **평어체 (반말)** 를 사용할 수 없고 맞춤법에 어긋나는 표현 혹은 신조어는 포함되지 않아야해요.
* 특수 문자를 반복 사용하거나 과도한 강조나 과장 표현으로 오해 될 수 있는 경우 사용을 제한하고 있어요.
  * <mark style="color:orange;">사용 불가 예시)</mark>&#x20;
    * \[함께해요.] \[만나보세요\~]&#x20;
    * \[어서오세요\~!!!]
    * \[만두]와\[돈까스]를 특가! 할인!
*   **TIP**&#x20;

    유저가 흥미를 가질 수 있는 하나의 혜택을 강조하여 작성해주세요.

{% tab title="이미지" %}
<mark style="background-color:red;">메인  타이틀과 함께 유저에게 이미지로 빠르게 브랜드/상품을 인식시킬 수 있는 영역이에요.</mark>

* 사용 가능한  이미지 사이즈는 1500\*760이에요.
* **가독성을 위해 이미지 내에 텍스트는 최대한 사용되지 않도록 구성해주세요.**
  * 브랜드 로고의 경우 상단 좌측 또는 우측에 위치할 수 있도록 구성해주세요.
  * 디자인 요소의 텍스트는 가능하지만, 정보 전달성 텍스트 사용은 불가해요.
* 이미지 내에 여백을  확보하여  구성해주세요. **(최소 20% )**
  * 단색 배경 외 상품 이미지, 텍스트, 꾸밈 요소, 배경 패턴  등은 여백으로 판단하지 않아요.
* 배경이 없거나 흰색 또는 검정색 배경을 사용한 이미지는 사용이 불가해요.
  * 유저가 설정한 토스 앱 화면 모드에 따라 이미지 노출이 원활하지 않을 수 있어요.
* 지나치게 채도가 높은 이미지는 사용할 수 없어요.

### 2. 액션 영역

<figure><img src="../../.gitbook/assets/액션 영역.png" alt=""><figcaption></figcaption></figure>

{% tabs %}
{% tab title="이벤트 안내" %}
* **문의처 (브랜드명)**  외에 내용은 수정 불가해요.
  * 추가 안내가 필요한 경우 광고 심의를 위해 꼭 필요한 정보만 입력해주세요.
* 기본 입력된 내용은 아래와 같아요.
  * 자세한 사항은 연결된 이벤트 안내 페이지에서 확인해주세요.
  * 이벤트 관련하여 궁금하신 사항은 (브랜드명) 고객센터에 문의해주세요.
  * 머니알림 컨텐츠 및 이벤트 내용, 토스 회원과 제휴사 간의 관계에서 이루어지는 거래 행위, 회원가입및 기타 행위에 대해서는 이벤트 주체인  제휴사에  책임이 있어요.
* 사전 심의 업종 금융상품 및 의료 광고는 심의필 번호를 기재해주세요.
{% endtab %}

{% tab title="CTA 액션 버튼" %}
* CTA 버튼명은 \[**액션 + 10원 더 받기]** 같은 형태로 기재해주세요.
* 글자 수는 공백 포함 최대 6자까지 작성 가능해요.

<mark style="background-color:red;">예시 )</mark>&#x20;

* ㅇㅇ 방문하고 10원 더 받기
* ㅇㅇ 확인하고 10원 더 받기
* ㅇㅇ 구경하고 10원 더 받기
{% endtab %}

{% tab title="CTA 버튼 연결 URL" %}
* 완성된 **웹사이트/앱** 으로 랜딩되는 링크를 사용해주세요.
  * 앱 페이지로 랜딩될 경우 앱을 설치하지 않은 유저는 앱스토어/구글스토어로 랜딩 되어야 해요.
  * 앱을 설치한 유저는 앱 안으로 연결되어야 해요.
  * 랜딩 과정에서 접속하는데 5초 이상 걸리는 경우, 브릿지 페이지를 만들어야해요.
* **유저의 기기(iOS/Android 등)와 관계 없이 랜딩 페이지로 정상연결되어야 해요.**
  * 일부 기기에서만 집행하는 광고는 지양하고 있어요.
* **URL 정책**
  * 주소 내 한글이 들어가 있는 경우 오류가 발생할 수 있어 반드시 영문 URL을 사용해주세요.
  * 링크 내에 **{ } ^ #** 문구가 있는 경우 광고 집행 불가해요.
    * **^** 대신 **\_**(언더바) 사용 시 승인 가능해요.
  * http:// 는 사용이 불가능하여 **https://** 를 사용하여 URL을 구성해주세요.
  * 비틀리와 같은, **숏링크/단축 URL형태는 승인이 불가해요.**
    * 앱스플라이어로 생성한 원링크 기입 시에도 축약형 숏링크가 아닌 원본 롱링크를 기입 해주세요.
* 이외의 사유로 CTA 링크에 이슈가 있다고 판단되는 경우, 링크 수정 안내 드릴 수 있어요.
{% endtab %}

{% tab title="랜딩 페이지" %}
* 광고소재에 기재된 **제품, 이벤트, 혜택 등 주요 내용이 랜딩 페이지에서 확인되지 않는 경우** 광고 집행이 불가해요.
  * 랜딩 페이지 수정이 어렵다면, 팝업이나 고정된 배너를 활용하여 광고 소재에 기재한 혜택 정보를 기재해 주세요.
  * 저작권 침해와 같이 이슈 발생 가능성이 있는 소재는 광고 집행이 불가해요.
  * 광고를 접한 유저에게 유익한 이벤트, 혜택 정보를 제공하는 내용으로 구성해주세요.
* **랜딩 페이지 심사 규정은 아래와 같아요.**
  * **모바일 기준으로 광고가 집행되니,** 모든 랜딩 페이지는 반드시 모바일 전용 웹/앱 페이지로 구성되어야 해요.
    * 단, 앱 설치 목적의 광고에 한해 앱스토어/구글 스토어로 랜딩 되는 것을 허용 가능해요.
  * 광고의 정보를 바로 제공하지 않고 제약 조건(**로그인 혹은 회원가입**) 선행되는 랜딩페이지는 사용이 불가해요.
  * 랜딩 페이지 내에는 접속한 유저가 상품/이벤트/프로모션에 대해 문의할 수 있는 문의처가 기재되어 있어야 해요.
  * 랜딩 페이지는 **1개의 생성 광고 당 1개의 URL만 사용 가능**해요. 사용하고자 하는 랜딩 페이지가 2개인 경우 새로운 캠페인을 생성해주세요.
  * 과도한 팝업으로 광고의 내용이 보이지 않는 경우 광고 집행이 어려워요.
* 자사와 공식적으로 제휴된 광고가 아닌 경우 \[토스] 의 브랜드명, 서비스명은 사용이 불가해요.
  * 내부 협의가 완료된 경우 증빙 자료를 제출해주세요.
{% endtab %}
