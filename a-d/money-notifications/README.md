# 머니알림

> **\[기본형 타입]** 머니알림은 최근 소비 이력을 기반으로 고객 페르소나를 특정하여 메시지를 발송
>
> **\[키워드형 타입]** 카드 알림 동의 유저 대상으로 특정 결제가 일어난 순간에 발송하여 즉각적인 액션을 유도
>
> **\[라이브형 타입]** 라이브 방송 시작 시간에 맞춰 타겟 유저들을 방송으로 인입 유도 &#x20;

<figure><img src="../../.gitbook/assets/머니알림 소개.png" alt=""><figcaption></figcaption></figure>

***

* **원하는 광고 효과에 맞게 유형을 선택하고 소재를 생성해주세요.**

### 광고 생성을 위해 각 항목에 맞는 정보를 입력해주세요

<figure><img src="../../.gitbook/assets/광고 유형 설정_기본 (1).png" alt=""><figcaption><p>광고 유형</p></figcaption></figure>

✔ **광고 유형:** \[일반] , \[라이브] 중에 선택 가능해요.

* 일반: 최근 소비 이력 기반으로 타겟 유저에게 발송&#x20;
* 라이브: 라이브 방송이 시작될 때 타겟 유저에게 발송&#x20;

<figure><img src="../../.gitbook/assets/광고 타겟-1.png" alt=""><figcaption><p>광고 타겟</p></figcaption></figure>

✔ **광고 타겟:** 토스의 머니 알림 구독자 대상으로 원하는 광고 메세지를 보낼 수 있어요.

* 새 타겟 설정하기 : 처음부터 조건을 설정하여 새로운 타겟 생성
* 생성된 타겟 선택하기 : 기존에 생성해둔 타겟이 있다면 불러와서 생성
  * 설정한 타겟 정보에 따라 기본 CPP 단가에 할증이 적용될 수 있어요 <mark style="background-color:red;">**\[기본 CPP 단가 : 30원]**</mark>
  * 타겟 조건 추가 및 설정에 따라 각각 10원씩 할증 적용되며 최대 100원까지 할증 될 수 있어요.
  * 연령, 성별, 지역, 디바이스 타겟팅은 할증 적용되지 않아요.
  * 광고 집행을 위한 **최소 타겟 규모는 3만명**이에요.

✔ **예상 타겟 정보**: \[타겟 정보 조회하기]를 누르면 광고 메세지를 보낼 수 있는 **예상 타겟 규모와 비용**을 확인할 수 있어요.
{% endstep %}

### 광고 집행 일시 선택하기

<figure><img src="../../.gitbook/assets/집행 일시 (1).png" alt=""><figcaption></figcaption></figure>

✔**분 당 발송 속도:** 원하는 분당 발송 속도를 선택해주세요.

* 운영하는 서버가 작을 경우 5천 명으로 설정하는 것을 권장해요.

✔광고 집행 날짜: 선택 가능한 날짜만 활성화되며, 당일 집행을 불가해요.

✔집행 시간:  09시 -  21시 사이에만 예약  설정 가능하고 발송 소요 시간이 적을 수록 선택 가능한 시간대가 많아져요.

* 원하는 시간대가 없다면 \[예약 가능한 모든 집행 일시 보기] 를 눌러 예약 가능한 시간 대를 볼 수 있어요.
{% endstep %}

### 광고 생성을 위해 각 항목에 맞는 정보를 입력해주세요

✔ **예산:** VAT가 미포함된 금액으로, 최소 100만원부터 설정할 수 있어요.

✔ **노출기간:**&#xD55C; 달 단위로 선택 가능해요.

* 설정예시)
  * 4월 24일 시작 > 4월 30일 까지만 설정 가능&#x20;
  * 5월 2일 시작 > 5월 31일 까지만 설정 가능

<figure><img src="../../.gitbook/assets/광고 타겟 (1).png" alt=""><figcaption></figcaption></figure>

✔ **광고 타겟 :** 설정 가능한 타겟 조건은 **연령**과 **성별**이고 그 외 조건은 추가할 수 없어요.

✔**키워드:** 선택한 키워드가 포함된 결제가 발생하는 유저에게 광고 메세지를 발송해요.

* 키워드는 최소 5개 최대 20개까지 선택 가능해요.
* <mark style="background-color:red;">**\[키워드 리스트 보기]에서 제공하는 키워드가 최소 3개 이상 포함되어야 광고 집행 가능해요.**</mark>
  * **키워드 작성 TIP**
    * 검색을 통해서 나오는 키워드를 활용해야 더 정확하게 타겟할 수 있어요.
    * 직접 입력하는 경우, 포괄적인 단어 또는 브랜드 키워드를 활용해주세요.
      * **검색 예시)**&#x20;
        * 정형외과, 주유소 → ( O)&#x20;
        * 산본정형외과, 평택햇살주유소  → (X)

{% hint style="info" %}
**키워드형 머니알림은 언제 발송되나요?**&#x20;

* 토스에서 카드 알림 수신을 동의한 유저 대상에게 결제 후 5분 이내 발송돼요.
* 유저가 카드 알림을 동의하지 않았을 경우, 해당 유저가 직전 7일 이내 광고주가 선택한 키워드 결제 이력이 있을 시 머니알림이 발송돼요.
* 복수의 키워드를 결제한 유저에게 머니알림이 중복 발송될 수 있어요.
{% endhint %}
