# 소재 가이드

## 신용점수 보드 소재 공통 가이드

* 소재 검수 시간은 **평일 10:00 \~ 19:00**이며, 모든 소재는 순차적으로 검수되고 있어요&#x20;
  * <mark style="color:blue;">원활한 소재 검수 및 광고 운영을 위해서 집행일 기준</mark> <mark style="color:blue;"></mark><mark style="color:blue;">**2-3영업일 전**</mark> <mark style="color:blue;"></mark><mark style="color:blue;">소재를 제출해주세요.</mark>
* **파일명은 영문 혹은 숫자로만 구성되어야 해요.**
  * ex) toss\_videoads\_2305 (O)
  * ex) 토스\_이미지\_최종 (X)
* 기재된 정보가 랜딩 페이지에서 확인되지 않거나, 허위 사실로 밝혀질 경우 광고 집행 불가 / 이미 승인되었더라도 취소될 수 있어요.
* 문구 색상 변경 옵션을 제공하지 않아요.
* 이모티콘은 사용할 수 없어요.
  * 예) ^\_^ / ★ / 😀/ :) / ㅠㅠ
* 사용 가능한 특수 문자는 다음과 같고, 중복 사용은 불가능해요.
  * % & - + = · ?, / → ‘’ “” \~ !
* **자사와 공식적으로 제휴된 광고가 아닌 경우, ‘토스’ 브랜드명, 서비스명을 사용하실 수 없어요.**

* **유저가 접속하는 데 이상이 없어야 해요.**
* **랜딩/이벤트 페이지의 내용과 소재 내용이 일치해야 해요.**
  * **이벤트 페이지에 없는 내용은 소재에 작성할 수 없어요.**
  * 광고의 정보를 바로 확인할 수 없고, 제약 조건 (로그인 또는 회원가입)이 선행되는 경우 집행이 어려워요.
* **URL 주소는 반드시 영문으로 전달해주세요.**
  * 주소에 한글이 있는 경우 오류가 발생할 수 있어요.
  * 링크 내에 `{ } ^ #` 문구가 있으면 반려될 수 있어요.
    * `^` 대신 `_`(언더바)를 사용해주세요.
  * http:// 는 사용이 불가하여 `https://` 를 사용해주세요.
* **앱스플라이어로 생성한 원링크를 제출할 경우, 파라미터가 단축되지 않은 원본 롱링크를 제출해 주세요.**
* **숏링크/단축 URL 형태는 사용할 수 없어요.(비틀리 포함)**
* **이외의 사유로 CTA 링크에 이슈가 있다고 판단되는 경우, 링크를 재요청 할 수 있어요.**
