# 광고 머니 관리

{% hint style="info" icon="folder-check" %}
## \[운영 정책 변경 사전 안내]

* **2026년 4월**부터 광고비 관리 체계가 '[비즈니스 월렛](https://business-wallet.gitbook.io/guide)'으로 이관됩니다.
  * 이에 따라 기존 **'광고머니'는** '**비즈니스 머니'**&#xB85C; 명칭이 변경되며, 향후 광고비는 비즈니스 월렛을 통해 통합 운영될 예정입니다.
  * 안정적인 서비스 운영을 위해 이관은 순차적으로 진행됩니다.&#x20;
* 자세한 운영 정책 및 이용 방법은 공지사항을 통해 안내드릴 예정입니다.&#x20;
{% endhint %}

### 광고머니란?

> 광고머니는 toss ads에서 광고 집행 시, 광고 비용을 결제하는데 사용

**유상 머니**

* 신용카드, 가상계좌 결제를 이용해서 직접 충전 후 사용하는 광고머니&#x20;
* **후불 정산 계정**의 경우, 별도 유상머니 충전 없이 후불 한도 내에서 광고를 먼저 집행하고 집행된 유상머니 금액에 대해서 **익월 초에 일괄 정산**하게 됩니다.

**무상 머니**

* 토스에서 **프로모션**, **이벤트 진행** 등으로 인해 지급한 무상 광고머니를 의미해요.
* 무상머니는 지급 시 설정된 만료일까지만 사용이 가능하며, 만료 기한이 지나면 **자동으로 소멸**돼요.
* 무상머니와 유상머니가 모두 있는 경우, 만료일 기한이 가까운 무상머니부터 **자동**으로 우선 **소진**되며, 특정 캠페인에만 사용할 수 없어요.

***

### 광고머니 관리하기

**\[광고머니 관리]** 메뉴에서 소진된 광고머니 내역을 확인하고 관리하고 충전 / 환불 이용할 수 있어요.

<figure><img src="../.gitbook/assets/Group 254.png" alt=""><figcaption></figcaption></figure>

* 광고머니  \[충전], \[소진], \[환급], \[환불] 버튼을 눌러 상세 내역을 볼 수 있어요.
  * 당일 발생한 충전, 소진, 환불, 내역의 합계와 잔액은 실시간으로 반영되지 않아 일치하지 않을 수 있어요.
* 자동 충전 기능을 활성화 하지 않은 경우 충전된 광고머니가 소진되는 경우 집행 중인 광고가 자동으로 중단돼요.
* 자동으로 중단 반영되는 기간 동안에 유효한 광고 클릭 혹은 집행이 발생하는 경우, 마이너스 잔액이 발생할 수 있어요.

***

### 광고머니 충전하기

1. \[광고머니 관리] 메뉴 →  상단 \[충전] 버튼 클릭

<figure><img src="../.gitbook/assets/맞춤 타겟_업종 (2).png" alt=""><figcaption></figcaption></figure>

3. 결제 완료 후, 총 광고머니 영역 또는 충전 내역 확인

### 광고머니 자동 충전하기&#x20;

광고머니가 소진되면 자동으로 광고 집행 중단되는 것을 원하지 않는 경우 광고머니 자동 충전 기능을 활성화 하는 것을 권장해요 예산에 맞춰 충전 조건과 금액을 설정할 수 있어요.

<figure><img src="../.gitbook/assets/6) 맞춤 보고서_기간-2 (1).png" alt=""><figcaption></figcaption></figure>

* 결제 수단은 **계정장 권한**을 가진 구성원만 <mark style="background-color:blue;">등록 / 변경 / 삭제</mark> 할 수 있고 **구성원 권한**을 가진 경우 <mark style="background-color:blue;">자동 충전 조건/ 카드 정보</mark>만 조회할 수 있어요.
* 광고 계정 단위로 한개의 카드만 등록 할 수 있고 추가 등록은 어려워요.

<figure><img src="../.gitbook/assets/맞춤 타겟_업종-1.png" alt=""><figcaption></figcaption></figure>

* 결제 수단은 개인 카드와 **법인 카드** 중 선택하여 등록할 수 있어요.
* **충전 조건**은 최소 1만원 이하 부터 최대 50만원 이하까지 설정 할 수 있어요.
* **충전 금액**은 최소 10만원 부터 500만원 까지 선택할 수 있고 직접 입력 한도는 일 최대 한도 1000만원을 초과할 수 없어요.
* 카드 정보가 변경되거나 분실/정지 되는 등 정상 결제가 불가한 경우 자동 충전은 OFF 돼요.

{% hint style="info" %}
자동 충전 이용 중 문제가 발생되거나 문의 사항이 있는 경우 [채널톡](http://toss-business.channel.io/)으로 문의 주시면 빠르게 도움 드리고 있어요.
{% endhint %}

### 광고머니 환불하기

* 충전 후 사용하지 않은 **유상 광고머니**에 대해 환불 요청할 수 있어요.
  * 환불 요청은 영업일 기준 최대 3일 이내 신청이 접수 되어 토스애즈 관리자 검토 후 처리돼요.
    * 환불 신청 후 결제 수단에 따라 입금 완료 (결제 취소) 까지 **영업일 기준 3-5일** 소요&#x20;
      * 거래일 기준 12개월 이내 결제 건에 한 해 취소 가능 (12개월 경과 시 카드사 및 결제 정보에 따라 정상 환불이 어려울 수 있어요.)&#x20;
* 각 광고 계정의 **계정장 권한**을 가진 구성원만 환불 요청이 가능해요.

<figure><img src="../.gitbook/assets/6) 맞춤 보고서_기간-1 (2).png" alt=""><figcaption></figcaption></figure>

* '**집행 중'**&#xB610;는 **'집행 완료'**&#xB41C; 모든 광고의 비용이 확정된 이후 환불 요청을 할 수 있어요.
* 광고 머니는 충전한 순서대로 금액이 소진되어 결제 방법과 소진 순서에 따라 환불 방법이 정해져요.

{% hint style="warning" %}
광고머니 충전 영수증은 아래 토스페이먼츠 [결제내역 조회 ](https://consumer.tosspayments.com/payment-history)페이지에서 직접 조회 가능해요.
{% endhint %}
