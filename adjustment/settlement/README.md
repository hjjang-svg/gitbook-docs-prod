# 정산하기

## 정산서 조회&#x20;

* 매월 1일 부터 말일까지 실제 집행된 광고에 대해 **유상 광고머니** 소진 금액을 기준으로 **익월 1일**에 정산서가 생성됩니다.&#x20;
  * 비즈니스 그룹의 \[그룹 관리 > 정산 메뉴]에서 각 광고 계정의 정산 내역을 확인할 수 있습니다.
  * 광고계정의 \[세금계산서] 메뉴에서 상세 정산 내역을 확인할 수 있습니다.&#x20;

<figure><img src="../../.gitbook/assets/비즈 그룹 정산.PNG" alt=""><figcaption><p>비즈니스 그룹 정산 메뉴</p></figcaption></figure>

## 세금계산서 발행

* 광고 계정 생성 시 등록한 **사업자 등록번호**로  매월 1일 \~ 말일에 실제 집행된 광고에 대해, 유상 광고머니 소진금을 기준으로 **월 1회** 세금계산서가 발행됩니다.
  * 무상 광고머니 사용 금액에 대한 내용은 포함되지 않습니다.

{% hint style="info" %}
- 발행 금액 : 월 1일 \~ 말일에 실제 집행된 광고에 소진한 **유상 광고머니** 비용&#x20;
- 발행 기준 : 광고계정 단위로 1장 발행
- 발행 주기 : 익월 초 (영업일 기준 4일 이내)
- 발행 대상 : 해당 월에 유상 광고 머니를 이용해 광고를 집행한 이력이 있는 광고 계정
- 발행 일자 : 해당 월의 말일
- 수신 이메일주소 : 광고 계정의 **대표 이메일** 주소
{% endhint %}

※ 일부 에이전시 계정의 경우 **사전 협의**를 통해 사업자 등록번호 단위로 1장 발행할 수 있습니다.

※ 세금계산서 수정 발행이 필요한 경우, [bo.team@toss.im](mailto:bo.team@toss.im) 메일로 문의 주시면 확인 후 순차적으로 도움 드리고 있습니다.
