# 소재 가이드

### 두근두근 1등 찍기 소재 설정 가이드

<figure><img src="../../.gitbook/assets/두두찍_영역 안내 (1).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/두두찍_영역 안내-1.png" alt=""><figcaption></figcaption></figure>

* **소재 공통 가이드에 따라 메인/서브 카피에 평어체(반말)을 사용할 수 없어요.**
* **랜딩 페이지에서 확인 불가한 혜택 및 이벤트 정보는 소재 내에 기재할 수 없어요. 반드시 연결되는 랜딩 페이지에서 확인 가능한 정보로 소재를 구성해주세요.**
* 소재 검수 시간은 **평일 10:00 \~ 19:00**이며, 모든 소재는 순차적으로 검수되고 있어요.

{% hint style="info" %}
- 소재 심사 가이드에 따라 심사 기간이 소요될 수 있어 광고 집행일 기준 2-3 영업일 전 소재를 제출해주세요 광고 집행일 기준 1영업일 전 또는 몇 시간 전 제출된 소재는 광고 집행 전까지 승인을 보장할 수 없어요.
- 두근두근 1등 찍기는 광고 집행 당일 소재 생성이 불가해요.
- 집행 상태가 \[집행 예정], \[집행 중], \[집행 완료], \[집행 중단] 이거나 \[심사 중], \[승인] 상태이 경우에는 소재 수정 및 집행 취소는 불가해요.
- 그 외 소재 수정이 필요한 경우 소재 집행 중단 처리 후 소재를 새롭게 제출해주세요.
  * **집행 중단 일자에 따라 위약금이 부과될 수 있으니 유의해주세요.**
{% endhint %}

***

* 투표 질문과 상품, 랜딩 페이지 모두 **연관성**과 **통일성**을 가지고 있어야 하고 연관성 없는 구성으로 확인되는 경우 심사 반려될 수 있어요.
* [**업종 별 심사 가이드**](https://toss-ads.gitbook.io/guide/policy/industryguide)에 따라 심의필 제출이 필수인 업종의 경우, 카테고리를 선택하고 심의필 문구 작성란에 기재해주시면 하단 영역에 자동으로 반영돼요.

{% tabs %}
{% tab title="투표 질문지" %}
- 투표 질문지는 **\[추천 문구]**&#xC640; **\[직접 입력]** 중 선택하여 구성할 수 있어요.
  * 추천 문구 : 유저의 반응이 좋았던 질문을 추천해요.
  * 직접 입력:한줄당 14자, 최대 2줄까지 가능하고, 한 문장으로 읽히게 작성해주세요.
    * 문장은 반드시 **은?/는? 형식의 의문형**으로 끝나야 해요.
    * &#x20;**? (물음표) , (쉼표)** 외에 특수 문자는 사용할 수 없어요.
- 지나치게 일반적이거나 모호한 표현 또는 줄임말은 사용할 수 없어요.
- 유저가 투표를 하면서 쉽게 공감하고 행동을 상상할 수 있도록, **구체적인 상황을 제시**하는 것이 중요해요.
  * _**장기 휴가 여행 갈 때 꼭 필요한 것은?** → 유저가 여행을 준비하는 상황을 상상하도록 유도_   &#x20;
  * _**자취를 시작하는 친구에게 있으면 좋은 것은?  →** 가구 구매를 고려하는 유저의 상황을 반영_
- **브랜드명** 또는 **상품명**을 언급한 주제는  광고 효율을 위해 사용하지 않는 것을 권장해요.
  * 금융 업종에 한해 **브랜드명** 또는 **상품명** 기재가 광고심의 필수 조항인 경우, **관련 증빙자료** 제출이 필요해요.
{% endtab %}

{% tab title="상품 작성" %}
* **상품 작성**
  * 상품은 최소 4개 최대 10개 등록 가능해요.
  * 상품명은 공백 포함 최대 14자 입력 가능해요.&#x20;
  * &#x20;, (쉼표) 외에 특수 문자는 사용 불가해요.
* **유의사항**&#x20;
  * 상품 갯수가 많을 수록 유저의 관심도가 올라가요.
  * 설정한 상품의 순서와 상관 없이 유저에게 노출 시 랜덤으로 보여져요.
    * 상품명 내에 순위를 기재하는 경우 심사 반려될 수 있어요.
  * 중복되는 상품은 등록할 수 없어요 광고 효율을 위해 다양한 상품으로 구성해주세요.
  * 직접적인 금액대 기재 또는 현금성 리워드 상품은 등록 불가해요.
    * <mark style="color:orange;">사용 가능한 상품명 예시)</mark>
      * <mark style="color:$info;">갤럭시 Z 플립5</mark>
      * <mark style="color:$info;">다이슨 에어랩</mark>
      * <mark style="color:$info;">스타벅스 아이스 아메리카노</mark>
      * <mark style="color:$info;">유럽 여행권</mark>&#x20;
      * <mark style="color:$info;">000 상품권</mark>
      * <mark style="color:$info;">호텔  숙박권</mark>
    * <mark style="color:orange;">사용 불가한 상품명 예시)</mark>
      * <mark style="color:$info;">💥핫딜💥 갤럭시 Z 플립5 (특수문자 포함 → X)</mark>
      * <mark style="color:$info;">\[HOT] 다이슨 에어랩 (특수문자 포함 → X)</mark>
      * <mark style="color:$info;">문화 상품권 3만원 (직접적인 금액대 언급 →  X)</mark>
      * <mark style="color:$info;">000 포인트 5만원  (현금성 리워드/금액대 언급 →  X)</mark>
      * <mark style="color:$info;">캐시백 2만원  (현금성 리워드/금액대 언급 →  X)</mark>
      * <mark style="color:$info;">1등 상품 \[명품백]  (순위  기재/특수문자 포함 →  X)</mark>

-   [x] 투표 질문과 연관된 상품과 광고 이미지로 구성해주세요.

    <mark style="color:$info;">-상품 이미지와 상품명이 상이한 경우 심사 반려될 수 있어요.</mark>

* **상품 이미지**&#x20;
  * **1:1 비율** (정사각형), **1024\*1024 사이즈** jpg, png만 쓸 수 있어요.
* &#x20;주요 제품이 명확하게 보이는 가독성 높은 이미지를 사용해주세요.
  * 테두리나 마진이 있는 이미지는 사용 불가해요.
* **아래와 같이 광고 이미지의 가독성과 질이 떨어지는 경우 광고 집행이 제한되며, 수정을 요청할 수 있어요.**
  1. 식별하기 어려운 텍스트,로고,뱃지 등의 이미지를 사용하는 경우&#x20;
     1. 브랜드 로고 이미지만 사용하거나 배경 색과 식별이 불가한 텍스트를 사용하는 경우
  2. 불쾌감또는 혐오감을 유발하거나 소재와 연관성이 없는 이미지를 사용하는 경우
  3. 과도한 크롭,클로즈업 이미지를 사용하거나 주요 제품을 알아볼 수 없는 분할 이미지를 사용하는 경우
  4. 토스 앱의 서비스 또는 프로모션으로 오인할 수 있는 이미지를 사용하는 경우
{% endtab %}

{% tab title="연결 URL" %}
* [x] 랜딩 URL 은 상품을 확인할 수 있는 페이지를 사용해하고, **회원 가입/로그인 절차 없이** 이벤트 페이지로 랜딩되어 바로 상품을 확인할 수 있어야 해요.

- **모바일 기준**으로 광고가 집행되며 완성된 **웹사이트/앱** 으로 랜딩 되는 링크를 사용해주세요.
  * 앱 페이지로 랜딩될 경우 앱을 설치하지 않은 유저는 앱스토어/구글스토어로 랜딩 되어야 해요.
  * 앱을 설치한 유저는 앱 안으로 연결되어야 해요.
  * 랜딩 과정에서 접속하는데 5초 이상 걸리는 경우, 브릿지 페이지를 만들어야 해요.
- **유저의 기기(iOS/Android 등)와 관계 없이 랜딩 페이지로 정상 연결 되어야 해요.**
- 일부 기기에서만 집행하는 광고는 지양하고 있어요.
- **URL 정책**
  * 주소 내 한글이 들어가 있는 경우 오류가 발생할 수 있어 반드시 영문 URL을 사용해주세요.
  * 링크 내에 **{ } ^ #** 문구가 있는 경우 광고 집행 불가해요.
    * **^** 대신 **\_**(언더바) 사용 시 승인 가능해요.
  * http:// 는 사용이 불가능하여 **https://** 를 사용하여 URL을 구성해주세요.
  * 비틀리와 같은, **숏링크/단축 URL형태는 승인이 불가해요.**
    * 앱스플라이어로 생성한 원링크 기입 시에도 축약형 숏링크가 아닌 원본 롱링크를 기입 해주세요.
- 광고 소재에 기재된 **제품, 이벤트, 혜택 등 주요 내용이 랜딩 페이지에서 확인되지 않는 경우** 광고 집행이 불가해요.
  * 랜딩 페이지 수정이 어렵다면, 팝업이나 고정된 배너를 활용하여 광고 소재에 기재한 혜택 정보를 기재해 주세요.
  * 저작권 침해와 같이 이슈 발생 가능성이 있는 소재는 광고 집행이 불가해요.
  * 광고를 접한 유저에게 유익한 이벤트, 혜택 정보를 제공하는 내용으로 구성해주세요.
- **유저의 사용성 보호를 위해 랜딩 페이지 내에는 접속한 유저가 상품/이벤트/프로모션에 대해 문의할 수 있는 문의처가 기재되어 있어야 해요.**
- **자사와 공식적으로 제휴된 광고가 아닌 경우 \[토스] 의 브랜드명, 서비스명은 사용이 불가해요.**
  * **내부 협의가 완료된 경우 증빙 자료를 제출해주세**요.
- 이외의 사유로 CTA 링크에 이슈가 있다고 판단되는 경우, 링크 수정 안내 드릴 수 있어요.
{% endtab %}

{% tab title="브랜드 및 심의필" %}
* **브랜드명**
  * **공백 포함 최대 10자까지 입력 가능해요.**
    * 상품명이나 기획전명이 아닌 상품을 구매할 수 있는 판매처의 브랜드명을 기재해주세요.
* 랜딩 페이지 내에서 확인할 수 없는 브랜드명을 기재하거나 실제 판매처 정보와 상이한 경우 심사 반려될 수 있어요.
{% endtab %}
