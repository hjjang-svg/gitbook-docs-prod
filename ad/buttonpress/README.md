# 버튼 누르기

<figure><img src="../../.gitbook/assets/버튼 누르기 소개.png" alt=""><figcaption></figcaption></figure>

* [x] 버튼누르기 상품은 저렴한 단가로 유저를 획득할 수 있고 많은 트래픽을 확보하는 광고 상품이에요.

토스 혜택 탭 지면에서 배너 형태로 노출되며 클릭 시 포인트를 획득할 수 있는 CTA 버튼이 노출되고, 클릭 후 브랜드 랜딩 페이지로 이동할 수 있어요.

<p align="center">소재가 승인된 시점부터 광고 노출이 시작되니 원활한 광고 운영을 위해 집행일 <strong>기준 2-3 영업일 전 심사 제출</strong>해주세요.</p>

#### 광고 생성을 위해 각 항목에 맞는 정보를 입력해주세요

<figure><img src="../../.gitbook/assets/광고 생성하기 (7).png" alt=""><figcaption></figcaption></figure>

✔ 광고 집행 일시: 일 단위로 선택할 수 있고 비활성화 되어 있는 날짜는 선택 불가해요.

* 주중 : 하루 단위로 선택할 수 있으며, 19시00분 \~ 23시 59분 동안 광고가 노출돼요.
* 주말 : 경우 토요일 00시 \~ 17시 59분, 일요일 00시 \~ 17시 59분 동안 광고가 노출돼요.
