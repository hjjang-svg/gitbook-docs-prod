{% tab title="브랜드명,브랜드 이미지" %}
- [x] &#x20;**아이콘 영역**에 입력한 브랜드 정보와 무관하게 **광고 문구** 또는 **광고 이미지**에도 브랜드 정보를 반드시 기재해주세요.

* **브랜드명**
  * 공백 포함 최대 20자까지 작성할 수 있어요.
  * 랜딩 페이지에서도 확인 가능한 실제 <mark style="background-color:blue;">브랜드명</mark> 또는 <mark style="background-color:blue;">제품명</mark>만 입력가능해요.
  * 혜택 또는 정보 전달성 문구를 입력하는 경우 심사 반려될 수 있어요.
* [브랜드 이미지](logo.md) 설정 가이드를 참고하여 이미지를 설정해주세요.
  * &#x20;로고 이미지가 실제 노출 지면에 반영되었을 때 잘림 또는 왜곡 없이 잘 보여지도록 가이드를 참고하여 **이미지 여백**을 충분히 넣어주세요.

<figure><img src="../../../.gitbook/assets/로고,제품 안전 여백 (10).png" alt=""><figcaption></figcaption></figure>

* 광고 소재와 연관성이 있는 이미지를 사용해야 하고 단순 텍스트 및 기타 이미지는 사용 불가해요.
* 로고의 형태가 **긴 문구**로 구성되어 있는 경우 실제 지면에서 브랜드 로고의 가독성이 떨어질 수 있어 사용 권장하지 않아요,  로고가 최대한 중앙에 위치할 수 있도록 구성해주세요.

<figure><img src="../../../.gitbook/assets/로고 가로 세로-1 (1).png" alt=""><figcaption></figcaption></figure>

* 실제 광고 집행 시 브랜드 이미지는 가독성을 위해 앱 버전이 다크모드인 경우 하얀색 테두리가 생길 수 있어요.

{% tab title="광고 이미지" %}
* **1080px\*1920 픽셀 이상의 png 이미지**&#x20;
  * 이미지 내 오브젝트를 배치하실 경우 아래 여백 가이드를 참고하여 배치해 주세요.
    * 상하 : 150px /좌우 : 150px
  * 업종에 따라 광고 소재 하단에 경고 문구 기입이 필요할 경우 **좌우 150px, 상하 50px의 안전 여백**을 확보해 주세요.
  * 사용자 디바이스 설정이나 기종에 따라 이미지가 상하·좌우로 조금 잘려 보이거나 소재 미리보기 화면과 다르게 보일 수 있어요.
* 세로형 사이즈로 9:16 비율이 지켜져야 해요.
  * 이미지 용량은 최대 3mb까지예요.
* **이미지 색상 가이드**
  * 배경이 없거나 흰색 또는 검정색 배경을 사용한 이미지는 사용이 불가해요.
    * 유저가 설정한 토스 앱 화면 모드에 따라 이미지 노출이 원활하지 않을 수 있어요.
  * 강한 형광색, 혐오감을 유발하는 이미지 등 퀄리티가 낮은 이미지는 사용 불가해요.
* **이미지 내에 텍스트가 있다면 사용이 불가해요.**
  * 제품 내 각인된 복잡하지 않은 텍스트 경우 내부 심사 기준에 따라 확인 후 사용 승인하고 있어요.
* 이미지 내 피사체가 과도하게 많거나, 과도한 크롭 / 클로즈업으로 인해 주요 제품을 알아볼 수 없는 이미지는 사용 할 수 없어요.
* 3개 이상으로 분할되어 주요 제품을 알아볼 수 없는 이미지는 사용할 수 없어요.
* **토스 앱의 서비스 또는 프로모션으로 오인할 수 있는 이미지는 사용 불가해요.**
{% endtab %}

{% tab title="광고 문구" %}
* **주요 문구**
  * 공백 포함 한 행 당 최대 14자 입력 가능하고 2행 모두 사용해주세요.
    * 1행만 입력하여 심사 제출하는 경우 심사 반려될 수 있어요.
* **보조 문구**
  * 공백 포함 한 줄에 최대 18자 작성할 수 있어요.
* **버튼**&#x20;
  * 버튼 문구는 공백 포함 최대 7자까지 작성할 수 있고 영문, 특수문자는 사용 할 수 없어요.
  * 버튼 문구의 어미는 아래 옵션 내에서 활용할  수 있어요.
    * 보러가기, 확인하기, 알아보기, 구경하기, 구매하기, 사러가기, 보기, 설치하기, 다운로드, 받기, 앱 열기, 시작하기
  * 사용 가능 예시 )&#x20;
    * _기획전 보러가기_
    * _보험료 확인하기_
    * _신상품 구경하기_
    * _혜택 알아보기_
    * _할인 상품 보러가기_
{% endtab %}

{% tab title=" URL 연결" %}
* [**유저가 접속할 때 이상이 없어야 해요**](../../../policy/creativeguide.md#id-5)**.**
  * 앱으로 랜딩된다면, 앱 미설치 유저는 앱 스토어 / 구글 플레이스토어로 랜딩될 수 있어야 해요.
  * 랜딩 과정에서 접속하는 데 5초 이상 걸리는 경우, 브릿지 페이지를 만들어 주세요.
* [**랜딩/이벤트 페이지의 내용과 소재 내용이 일치해야 해요**](../../../policy/creativeguide.md#id-6)**.**
  * 랜딩 / 이벤트 페이지에서 확인할 수 없는 혜택, 정보는 광고 승인이 불가해요.
  * 가입 또는 응모와 같이 선행 조건이 따르는 혜택은 광고 문구 내에 반드시 명시해야만 사용할 수 있어요.
* [**URL 주소는 반드시 영문으로 제출해주셔야 해요**](../../../policy/creativeguide.md#id-4)**.**
  * 주소에 한글이 있는 경우 오류가 발생할 수 있어요.
  * 링크 내에 <mark style="color:red;">{ } ^ #</mark> 문구가 있으면 반려될 수 있어요.
    * <mark style="color:red;">^</mark> 대신 <mark style="color:red;">\_</mark>(언더바)를 사용해주세요.
  * <mark style="color:red;">http://</mark> 는 사용이 불가하여 <mark style="color:red;">https://</mark> 를 사용해주세요.
* 앱스플라이어로 생성한 원링크를 제출할 경우, 파라미터가 단축되지 않은 원본 롱링크를 제출해 주세요.
* 숏링크/단축 URL 형태는 사용할 수 없어요. (비틀리 포함)
* 이 외의 사유로 CTA 링크에 이슈가 있다고 판단되는 경우, 링크 수정 요청 할 수 있어요.
* 그 외 공통으로 적용되는 규정은 [소재 심사 규정](../../../policy/creativeguide.md#id-4) 을 참고해주세요.
