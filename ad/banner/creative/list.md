### 2-2. 연결 URL 가이드

<table><thead><tr><th width="262.140625">조건</th><th width="495.421875">내용</th></tr></thead><tbody><tr><td>유저가 접속할 때 이상이 없어야 함</td><td><ul><li>앱으로 랜딩 된다면, 앱 미설치 유저는 앱 스토어・플레이스토어로 랜딩</li><li>접속하는데 5초 이상 걸리는 경우, 브릿지 페이지 제작</li></ul></td></tr><tr><td>랜딩 / 이벤트 페이지의 내용과 소재 내용이 일치해야 함</td><td><ul><li>랜딩 페이지에서 확인할 수 없는 햬택 및 정보는 반려</li><li>가입, 응모와 같이 조건부 혜택은 광고 문구 내에 반드시 명시</li></ul></td></tr><tr><td>URL 주소는 반드시 영문으로 제출</td><td><ul><li>주소에 한글이 있는 경우에는 오류가 발생할 수 있음</li><li>링크 내에 <code>{ } ^ #</code> 문구가 있으면 반려될 수 있음</li><li>^ 대신 _(언더바) 사용 권장</li><li><code>http://</code>는 사용 불가 → <code>https://</code> 사용 권장</li></ul></td></tr><tr><td>URL 축약 불가</td><td><ul><li>앱플라이어로 만든 윈링크의 경우, 피라미터가 단축되지 않은 원본 제출</li><li>숏 링크 / 단축 URL 불가(비틀리 포함)</li></ul></td></tr><tr><td>기타</td><td><ul><li>이 외의 사유로 문제가 있는 경우, 연결 URL 수정 요청을 할 수 있음</li><li>그 외 공통으로 적용되는 규정은 <a href="https://toss-ads.gitbook.io/guide/policy/creativeguide">소재 심사 규정</a> 참고</li></ul></td></tr></tbody></table>
