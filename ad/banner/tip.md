# 유용한 기능

## 토스 앱에서 미리보기 &#x20;

휴대폰에서 QR 코드를 스캔하거나 링크를 복사해서 광고 소재가 토스 앱에서 어떻게 보이는지 광고 화면을 미리 확인할 수 있는 기능이에요.

<figure><img src="../../.gitbook/assets/미리보기 QR 예시.png" alt=""><figcaption></figcaption></figure>

### 사용 방법

{% stepper %}
{% step %}
#### 배너 광고 캠페인 생성&#x20;

<figure><img src="../../.gitbook/assets/미리 보기 위치.png" alt=""><figcaption></figcaption></figure>

* [x] &#x20;캠페인 생성 → 소재 설정 화면 우측 상단 \[미리보기] 버튼 클릭 →  \[토스 앱에서 미리보기] 버튼 클릭&#x20;
{% endstep %}

{% step %}
#### &#x20; 토스 앱에서 미리보기

<figure><img src="../../.gitbook/assets/미리보기 QR 예시 (1).png" alt=""><figcaption></figcaption></figure>

* &#x20;미리보기 기능은 아래와 같이 두 가지 방법으로 사용할 수 있어요.
  * 생성된 **QR 코드**를 휴대폰 카메라로 스캔&#x20;
  * **링크 복사** → 복사한 링크 모바일 환경에서 열어보기
{% endstep %}

{% step %}
#### 모바일 환경에서 소재 확인하기&#x20;

<figure><img src="../../.gitbook/assets/미리 보기 화면_예시-1.png" alt=""><figcaption></figcaption></figure>

* &#x20;토스 앱이 설치되지 않은 경우 미리보기 기능을 사용하기 어려워요.&#x20;
  * 미리보기 (토스 앱) 화면에서는 내 소재의 랜딩 페이지를 눌러보는 것 외에 다른 기능은 동작하지 않아요.
* 소재를 수정했다면 새로운 QR 코드와 링크를 복사해서 확인해주세요.

{% hint style="warning" %}
- 잠재고객 모으기 정보입력 양식은 미리보기 기능을 지원하지 않아요.&#x20;
- 랜딩 URL 유형에 따라 지원 방식이 달라요.&#x20;
  * 딥링크:  중괄호 { } 가 포함된 딥링크는 랜딩 페이지로 연결되지 않아요.
  * MMP: 미리보기 기능을 통해 접속한 경우, 성과 측정에서 제외되어요.&#x20;
{% endhint %}
{% endstep %}
{% endstepper %}







***





## 소재 가져오기&#x20;

소재 설정 화면에서 기존 소재들을 간편하게 살펴보고 소재명으로 검색해서 가져올 수 있는 기능이에요.

### 사용 방법

{% stepper %}
{% step %}
#### 배너 광고 캠페인 생성

<figure><img src="../../.gitbook/assets/소재 가져오기 (1).png" alt=""><figcaption></figcaption></figure>

* [x] &#x20;캠페인 생성 → 소재 설정 화면 우측 상단 \[소재 가져오기] 버튼 클릭&#x20;
{% endstep %}

{% step %}
#### 가져올 소재 선택하기

<figure><img src="../../.gitbook/assets/가져올 소재 선택하기.png" alt=""><figcaption></figcaption></figure>

* 기존 소재들을 한 눈에 살펴보고 필요한 소재를 선택해서 새로운 캠페인에 간편하게 불러올 수 있어요.
  * 소재명으로 검색하면 원하는 소재를 빠르게 찾을 수 있어요.
{% endstep %}

{% step %}
#### 잠재고객 모으기 정보입력 양식 불러오기&#x20;

&#x20;배너 소재와 동일하게 정보입력 양식도 기존 양식을 불러올 수 있어요 정보입력 양식 불러오기 경로는 아래와 같아요.

<figure><img src="../../.gitbook/assets/잠재고객 모으기 정보 입력 약식 불러오기 (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
* 일부 배너 유형은 소재 가져오기 기능을 지원하지 않아요.
* 가져올 수 있는 소재는 심사 결과에 상관 없이 **심사 신청**되었던 소재만 노출되고 **임시 저장** 상태의 소재는 노출되지 않아요.
{% endhint %}
